## 📌 Overview

This project simulates a **real-world fintech operations pipeline** used by trading desks and back-office systems.

It covers:

* 📩 Email ingestion from shared mailbox
* 🧠 Trade classification (Equity, FX, Derivatives)
* 🔍 Data extraction (Regex + AI concepts)
* 🛡️ Risk & compliance validation
* 📤 OMS (Order Management System) booking
* ⚠️ Error handling, retries & fallback system

## 🧩 Workflow Breakdown

### 🟡 Phase 01 — Email Ingestion

* Connect to mailbox API
* Parse email & attachments
* Retry logic with alert system

### 🔵 Phase 02 — Trade Classification

* Routes trades into:

  * Equity
  * FX Forward
  * Derivatives
  * ❌ Unknown → Manual Review

### 🟣 Phase 03 — Compliance & Processing

* Aggregate extracted data
* Build structured API payload
* Compliance engine results:

  * ✅ Approved → OMS Booking
  * ⚠️ Error → Diagnosis
  * ⏱ Timeout → Retry / Fallback

### 🔴 Phase 04 — Fallback System

* Store failed trades
* Notify compliance team
* Generate case for manual review
